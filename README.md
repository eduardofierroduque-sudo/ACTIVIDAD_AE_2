¡Hola! Este es un pequeño script que creé para automatizar el cálculo de notas de mis cursos imaginarios. Es ideal para estudiantes o profesores que necesitan un resultado rápido sin usar calculadoras manuales.
El programa te guía paso a paso:
Te pregunta el nombre del curso.
Te consulta cuántas notas quieres promediar.
Te pide cada nota una por una.
¡Listo! Te entrega el promedio final con un mensaje personalizado.
